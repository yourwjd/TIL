Strict mode (엄격 모드)

es5이후 수정된 기능 사용 시 이 모드를 통해 문제를 잡음

지시자 "use strict" 'use strict' 사용

사용하게 되면 모든 스크립트가 엄격모드로 실행

반드시 최상단에 위치, 그렇지 않으면 그 밑에 있는 부분만 엄격모드 실행됨

브라우저 콘솔에서 사용하려면 콘솔창에 'use strict'; 작성 후 줄바꿈(shift+enter)하면 작동함

코드를 클래스와 모듈을 사용하여 구성하면 생략해도 됨



2-4

변수

변수: 데이터를 저장하는 '이름이 붙은 저장소'

let을 사용

let message; 변수 선언
message = 'Hello'; 데이터 저장(값 할당)
alert('message'); 저장된 데이터 보여줌

한 줄에 변수 선언, 값 할당을 동시에 작성 가능함
let message = 'Hello';

한 줄에 여러 변수를 선언 가능함 (권장x)
let user = 'John', age= 25, message = 'Hello';

하지만 가독성을 위해 1줄 1변수 권장

값을 다름 변수의 값으로 복사할 수 있음
let Hello = "Hello World!";
let World; 복사할 변수
World = Hello; 값 복사

변수는 한 번만 선언 가능함
값을 변경하는 재할당은 할 수 있지만 let Hello;라고 이미 선언한 경우에는 Hello라는 변수는 재선언할 수 없음

변수명 규칙

  1. 문자, 숫자, 기호 $ _ 만 사용 가능
  2. 첫 글자에는 숫자가 올 수 없음
  3. 예약어는 사용할 수 없음
  4. 주로 카멜 표기법을 사용함
    ex) MyVeryLongName
  5. 대소문자를 구별함
  6. 영어 사용 권장

상수

상수: 변하지 않는 변수

const를 사용

상수는 변하지 않기 때문에 값을 바꿀 수 없음 = 재할당 불가

const myBirthday = '02.10.13'
이 상수 myBirthday 의 값은 02.10.13으로 고정

변하지 않을 값이라면 변수let 대신 상수 const를 쓰기

기억하기 힘든 값을 대문자 상수에 할당해 별칭으로 사용할 수 있음
const COLOR_RED = "F00"; 기억하기 힘든 컬러코드를 대문자 상수에 저장
let color = COLOR_RED; 상수를 별칭으로 사용
alert(color); 컬러코드를 쉽게 찾을 수 있음



2-5

자료형

값은 항상 문자열, 숫자형 등 특정한 자료에 속해있음

변수는 자료형과 관계없이 모든 데이터일 수 있음

let message;의 message는 문자열일 수도 숫자형일 수도 있음
이렇게 언제든지 바꿀 수 있는 변수에 저장되는 값의 타입을 동적 타입언어라고 함


  1. 숫자형 number type

    2. 정수 및 부등소수점 숫자

    연산은 *: 곱셈, /: 나눗셈, +: 덧셈, -: 뺄셈 ...

    특수 숫자 값 포함

    • Infinity, -Infinity

      • 어떤 숫자보다 더 큰 특수 값, 무한대
      어떤 숫자든 0으로 나누면 Infinity 즉 무한대를 얻을 수 있음

    • NaN

      • 부정확하거나 정의되지 않은 수학 연산 사용 시 에러 발생 즉 NaN 반환됨
      한 번 NaN이 반환된 이상 웬만해서는 변하지 않음
      연산 과정 중 반환 됐을 시 모든 결과에 영향을 미치게 됨/p>



  2. BigInt

    3. 253-1 (9007199254740991) 보다 큰 값 혹은 -(253-1) 보다 작은 정수는 숫자형으로 나타낼 수 없음

    BigInt 를 사용해 길이에 상관없이 정수를 나타낼 수 있음
    정수 리터럴 끝에 n을 붙이면 됨

    const bigInt = 1234567890123456789012345678901234567890n;



  3. 문자형 string type

    4. 문자열

    따옴표로 묶어 사용

    1. 큰따옴표: "문자열"
    2. 작음따옴표: '문자열'
    3. 역따옴표(백틱): `문자열`

    큰따옴표, 작은따옴표는 차이 x
    역따옴표를 사용하면 변수나 표현식을 $() 안에 넣어 표현할 수 있음

    let name = "John";
    alert(`Hello, ${name}!`) Hello, John! 출력

    alert(`the result is ${1 + 2}`); the result is 3 출력



  4. 불린형 logical type

    5. true 긍정, false 부정 값

    let nameFielsChecked = true;
    let ageFieldChecked = false;

    let isGreater = 4 > 1;
    alert( isGreater ); true 출력



  5. 'null' 값

    6. nothing 존재하지 않는 값, empty 비어 있는 값, unknown 알 수 없는 값

    오로지 null 값만 포함하는 별도의 자료형
    다른 언어의 null과 성격이 다름

    let age = null;



  6. 'undefined' 값

    7. 값이 할당되지 않은 상태

    let age;
    alert(age); undefined 출력

    직접 할당도 가능, 하지만 null 사용을 권장

    let age = 100;
    age = undefined;
    alert(age); undefined 출력



  7. 객체와 심볼

    8. 객체형 (object)

    데이터 컬렉션, 복잡한 개체 표현 가능


    심볼형 (symbol)

    객체의 고유한 식별자 만들 때 사용



  8. typeof 연산자

    9. 인수의 자료형을 문자열 형태로 반환

    typeof x 혹은 typeof (x) 형태로 사용

    typeof 0; "number" 출력
    typeof true; "boolean" 출력

    typeof null; 객체형(objrct)이 아니지만 하위호환성을 위해 오류를 출력
    typeof alert; 함수형(function)은 존재하지 않지만 하위호환성을 위해 출력

2-6

alert

사용자가 확인 버튼을 누를 때까지 메시지를 보여주는 창

alert("Hello");

메시지를 보여주는 창 = 모달 창(modal window)
'모달'이란 해당 페이지 외의 부분과 상호작용이 불가능하다는 의미

prompt

텍스트 메시지, 입력 필트, 확인/취소 버튼이 있는 모달 창

  1. title

    2. 사용자에게 보여줄 문자열

  2. default

    3. 입력 필드의 초기값(선택값)
    []대괄호는 필수가 아닌 선택이라는 의미
    취소, 자판 esc를 선택한 경우에는 null을 반환

result = prompt(title, [default]);

[default]는 선택값이지만 없는 경우에도 ''로 기본값을 넣어주는 것을 권장

confirm 대화상자

질문, 확인/취소 버튼이 있는 모달 창

result = confirm(question);

확인은 true, 취소는 false를 반환
