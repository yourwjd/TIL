형 변환 type conversion

함수, 연산자에 전달되는 값은 적절한 자료형으로 자동 변환됨

ex) alert의 문자열로 자동 변환, 수학 관련 연산자가 전달받은 값 숫자 변환 ...

명시적 변환: 전달받은 값을 의도를 갖고 원하는 타입으로 변환하는 경우

  1. 문자형으로 변환

    2. 문자형의 값이 필요할 때

    alert: 매개변수가 무조건 문자형이어야 하기 때문에 다른 형의 값을 전달 받으면 자동 변환
    String: 함수를 호출해 전달받은 값을 문자열로 변환

  2. 숫자형으로 변환

    3. 수학과 관련된 함수, 표현식에서 자동으로

  3. 불린형으로 변환

    4. 논리 연산을 수행할 때
